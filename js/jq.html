Working With Jquery

    jQuery is a JavaScript library designed to 
    simplify HTML DOM tree traversal and 
    manipulation, as well as event handling,
    CSS animations

    jQuery's syntax is designed to make 
    it easier to navigate a document, select 
    DOM elements, create animations, handle events

    jQuery also provides capabilities for
    developers to create plug-ins on top 
    of the JavaScript library.

    The modular approach to the jQuery 
    library allows the creation of
    powerful dynamic web pages and 
    Web applications.
